# Api-python
Proyecto Final de Programación Básica

En este proyecto se diseñó y programó una API REST que, a través de una serie de peticiones, guardaba cierta información proporcionada por el usuario (anime, vistas, ranking, títulos, sinopsis, casas de animación, licencias, estudios, etc.) y se generaba un archivo Excel y gráficas para explicar su contenido.

Se usaron diversas librerías como NumPy, Matplotlib, Pandas, etc. para la interpretación matemática y visual, y por otro lado las esenciales para hacer peticiones (requests) y guardarlas en un archivo JSON.
